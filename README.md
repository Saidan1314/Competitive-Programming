# Competitive-Programming
Optimized solutions to CSES, Project Euler and Codeforces problems encountered during my competitive programming journey.

### What is this project?
This project is a **collection** of solutions to non-trivial problems, focusing on **Mathematics, Combinatorics, Logic, Arithmetic, etc...**. 

Each file includes:
* **The Implementation:** Clean and optimized C++ code.
* **The Insight:** A detailed explanation of my approach, mathematical observations, and complexity analysis.
